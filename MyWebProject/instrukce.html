IP teploměr s digitálními a analogovými vstupy a výstupy

Vlastnosti

  1. Čidlo DS18B20 nebo Pt100 s převodníkem (nejprve však simulované)
  2. Zobrazení teploty v grafu za poslední den, týden, měsíc a rok.
  3. Data budou ukládána na SD kartě.
  4. Bude využito RTC timeru pro přesné časování. Možnost synchronizace času s počítačem.
  5. Vše nastavitelné pomocí webserveru. Přístup přes administrátorské heslo.
  6. Vše nastavitelné pomocí webserveru. Přístup přes administrátorské heslo.
  7. Aktuální stav teploty vstupů i výstupu se bude automaticky refreshovat (bez nutnosti načtení celé stránky).
  8. Stav vstupů i výstupů se bude znázorňovat graficky.
  9. Aplikace bude obsahovat API pro přímé ovládání pomocí spojového rozhraní TCP.
  10. Možnost nastavení tresholdů k analogovým a digitálním vstupům s reakcí odeslání emailu na požadovaný email.

Postup

  1. Vytvoření simulátoru v jazyce C#.

    1. Vytvoření formuláře pro přihlášení s použitím administrátorského hesla.
    2. Nadefinování stylu (barva pozadí, písma, nadpisy, rozmístění ...).
    3. Vytvoření stylované hlavní stránky s použitím HTML a CSS.
    4. Vytvoření simulovaného vstupně výstupního zařízení.
    5. Napojení hlavní stránky na simulované zařízení.
    6. Vytvoření API pro přímý přístup k simulovanému zařízení.
    7. Vytvoření aplikace pro automatické testování přímého API.

  2. Vytvoření aplikace na procesoru STM32 s OS FreeRTOS.

    1. Komunikace s en26j60. Odeslání jednoduchého paketu.
    2. Vytvoření UDP socketu pro vysílání.
    3. Vytvoření UDP socketu pro příjem.
    4. Vytvoření TCP socketu pro vysílání.
    5. Vytvoření TCP socketu pro příjem.
    6. Měření teploty pomocí čidla DS1820, ovládání vstupů a výstupů.
    7. Připojení SDcard a ukládání dat do souboru.
    8. Implementace web serveru - statické stránky.
    9. Implementace web serveru - plně funkční.
    10. Vytvoření api pro přímý přístup.
    11. Dokončení, správa napájení, dokumentace ...
